# Project Description
The proposed project, Seamless, is a product that is a web app designed for anyone interested in fashion and clothing. This web app allows users to explore their interests in clothing no matter who they are - whether someone wanting to try out new styles, someone who wants to track and save their outfits, or someone who wants to style for other people. This web app will allow users to upload images of themselves at various angles. With these images, AI will be used to scan these images and create a model of the user. The user can then upload images of articles of clothing to create an outfit, which can be saved in the virtual wardrobe of the user. Then, a picture is generated of the user wearing the outfit they desire to see. This is useful for users that want to see the outfit replicated exactly on them. 

# Major Features

## Upload and save image(s) of user
- When a user initially makes an account they are prompted to upload images of themselves from their devices to create a user “avatar” to be fed to the AI to be used as the “model” for the outfits. 
- Once the user has created their account they can go back into their user profile and delete images and upload new images to modify their avatar.

## Upload and save image(s) of clothing
- Users should be able to upload images of individual pieces of clothing or accessories they own to save to their Virtual Wardrobe, with various descriptive labels for each piece (“belt,” “top,” “spring/summer,” “formal,” etc.). 
- Within the Virtual Wardrobe the user should be able to edit these labels. The user should also be able to upload images of whole outfits which will be saved to their Virtual Wardrobe as an “outfit.”
- For pieces that the user does not own, but is just interested in, there will be a section of the Virtual Wardrobe called “Inspiration” to save clothes and outfits to.

## AI generated images of user in outfits
- The user will be able to select various items or a set outfit from their Virtual Wardrobe and generate (using an AI image generator) an image of themselves in that outfit.
- From there the user will have the opportunity to save that image to an area called Previous Looks, for future reference.
- The user will also be able to select from a list of styles and generate some images of themselves dressed in outfits of that style.

## Analysis
- The user will be able to ask the application for an analysis of what colors, styles, etc. would look good on them. (As if consulting a personal stylist.)
They will then have the option to generate some images of themselves in outfits matching this analysis.
